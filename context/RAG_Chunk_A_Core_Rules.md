# RAG Chunk A — Core Rules

- Profile consent ≠ suppression
- Consent = permission to send marketing
- Suppression = hard block on email marketing
- Suppression overrides consent
- Consent updates do NOT remove suppression
- profile-bulk-import-jobs:
  - create/update profile attributes only
  - cannot set consent
  - cannot modify suppression
- Transactional email/SMS must remain deliverable
- Do not suppress unless intentionally blocking email marketing
- “subscription” refers only to profile marketing consent (not lists)
